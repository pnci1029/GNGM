# GNGM (가는김에)

## 프로젝트 개요
**GNGM**은 "가는김에"의 줄임말로, 지역 커뮤니티 기반 배송 및 심부름 서비스 플랫폼입니다.

## 컨셉
당근마켓과 같은 동네 커뮤니티 개념을 기반으로, 누군가 어디를 가는 길에 다른 사람을 위해 물건을 사다주거나 사람/물건을 운반해주고 수고비를 받는 서비스입니다.

### 핵심 기능
- 🚗 **가는김에 배송**: 내가 가는 길에 다른 사람 물건 배송
- 🛒 **대신 구매**: 마트, 약국 등에서 물건 대신 사주기
- 👥 **동승 서비스**: 같은 방향으로 가는 사람과 카풀
- 💰 **수고비 시스템**: 간편한 결제 및 정산

### 서비스 특징
- **지역 기반**: GPS를 활용한 동네 단위 서비스
- **실시간 매칭**: 즉시 요청자와 제공자 연결
- **신뢰 시스템**: 평점 및 후기를 통한 사용자 신뢰도 관리
- **간편 결제**: 카카오페이, 토스페이 등 간편결제 연동

## 기술 스택

### Frontend (Mobile)
- **Framework**: Flutter
- **Language**: Dart
- **Platform**: Android, iOS

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Real-time**: WebSocket/Socket.io

### Infrastructure
- **Cloud**: AWS/NCP
- **Deployment**: Docker
- **Maps**: 네이버/카카오 지도 API
